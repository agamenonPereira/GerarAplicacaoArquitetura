# Gerar Aplicação Arquitetura
Gerar aplicação com base na arquitetura CQRS (Eduardo Pires)

Este é um projeto inicial que irá auxiliar o Desenvolvedor a produzir programas mais rápidos e padronizados.

# Como utilizar esta aplicação
Basta Abrir a aplicação (que é uma aplicação console) e informar os seguintes dados:
- Nome do Sistema
- Prefixos e Sufixos dos métodos da classe a ser gerada

A partir daí, será criado na pasta: C:\SistemaGerado, os projetos com classe e métodos abaixo:
- DomainCore
- Domain
- Application
- Infra.Data
- CrossCutting.Bus
- CrossCutting.IoC
- CrossCutting.Identity

e ainda gera a Solution com o nome do Sistema que foi informado.
